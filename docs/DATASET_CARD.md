# Описание набора данных

## Краткое описание

Набор данных содержит отзывы на аниме на русском языке. Данные предназначены для анализа тональности (sentiment analysis) и классификации эмоциональной окраски текста.

## Состав набора данных

### Исходная структура

Файл `Anime_reviews_RU.csv` содержит следующие столбцы:

1. **`Unnamed: 0`** — индекс записи (удален в процессе очистки)
2. **`Anime`** — название аниме (удален в процессе очистки)
3. **`Rate`** — оценка отзыва (Положительный, Нейтральный, Отрицательный)
4. **`Text`** — текст отзыва (ключевой признак для анализа)

### Размеры данных

- **Всего записей**: ~73,000
- **После очистки**: ~73,000 (удалены дубликаты и пропуски)
- **После предобработки**: ~73,000 (векторизованные тексты)

### Источники данных

Данные были собраны из различных источников, включая сайты аниме-сообществ и платформы для обмена мнениями. Датасет содержит:

- Тексты отзывов пользователей на аниме
- Оценки пользователей (положительные, нейтральные, отрицательные)

## Предобработка

В проекте используются три версии данных:

1. **`raw.csv`** — исходные необработанные данные (содержат дубликаты и пропуски)
2. **`cleaned_data.csv`** — очищенные данные, готовые для обучения
3. **`fasttext_format.txt`** — данные в формате fasttext для обучения векторизатора

## Стратегия разделения данных

Данные были разделены на обучающую и тестовую выборки в соотношении 80/20 с использованием функции train_test_split из sklearn с random_state=42:

- **Обучающая выборка**: ~58,400 записей
- **Тестовая выборка**: ~14,600 записей

## Лицензия

Данные предоставлены без лицензии. Использование ограничено только для исследовательских целей.
