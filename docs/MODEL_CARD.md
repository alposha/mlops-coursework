# Sentiment Analysis Model Card

## Назначение модели

Модель предназначена для классификации русскоязычных отзывов на три класса:

- **Положительный**
- **Нейтральный**
- **Отрицательный**

Используется для анализа отзывов пользователей на аниме и другие медиапроекты.

---

## Архитектура

- **FastText** — для векторизации текста.
- **CatBoostClassifier / RandomForest / LogisticRegression** — для предсказания класса на основе векторных представлений.
- В production используется CatBoost + FastText.

---

## Данные для обучения

- Данные подготовлены из очищенных отзывов (`cleaned_data.csv`), содержащих колонки:
  - `Text` — текст отзыва
  - `Rate` — метка сентимента

- Формат для FastText: `__label__<Rate> <Text>`.

---

## Метрики

Метрик для CatBoost на тестовом сплите (20% данных):

| Метрика   | Значение |
| --------- | -------- |
| Accuracy  | 0.8262   |
| Precision | 0.8259   |
| Recall    | 0.8262   |
| F1-score  | 0.8251   |

> Все метрики рассчитываются как **weighted average** для трёх классов.

---

## Ограничения

- Модель обучена на ограниченном датасете отзывов и может плохо работать на других доменах.
- Не учитывает сарказм и сложные контексты.
